## Overview
This project focuses on analyzing used car data in Mumbai, with a specific emphasis on Isuzu and Volvo cars. The goal is to uncover insights into pricing trends, customer preferences, and key market patterns using data cleaning, exploratory data analysis (EDA), and visualization techniques.

---

## Repository Structure
| File/Folder Name                        | Description                                                 |
|-----------------------------------------|-------------------------------------------------------------|
| `Car_list.xlsx`                         | Excel file containing additional car data.                  |
| `used_volvo_cars_in_mumbai.csv`         | Raw dataset of used Volvo cars in Mumbai.                   |
| `cleaned_used_volvo_cars_in_mumbai.csv` | Preprocessed dataset after initial cleaning.                |
| `further_cleaned_used_volvo_cars_in_mumbai.csv` | Refined dataset after further cleaning.              |
| `Isuzu_Cars_in_Mumbai_Project.ipynb`    | Jupyter Notebook for analyzing Isuzu cars in Mumbai.        |
| `LICENSE`                               | Apache-2.0 license for project usage.                       |
| `README.md`                             | Documentation for the project.                              |

---

## Features
1. **Data Cleaning:**  
   - Handling missing values and duplicates.
   - Standardizing data formats for consistency.

2. **Exploratory Data Analysis (EDA):**  
   - Visualizations including heatmaps, scatter plots, and bar charts.
   - Insights into pricing trends and feature correlations.

3. **Business Insights:**  
   - Recommendations for pricing strategies and feature prioritization.
   - Region-specific analysis to guide marketing efforts.
